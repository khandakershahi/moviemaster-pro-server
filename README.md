# ( সহজ সরল সিম্পল ) ASSIGNMENT-010

This project is the backend for the Movie Master Pro app built with Express for Programming Hero.

## Features

- User authentication with Firebase
- Movie search by title
- Add, update, delete movies
- Watchlist management
- Movie reviews
- User profile management
- Secure API endpoints
- MongoDB integration

## Technologies

- **NODE.JS**
- **EXPRESS**
- **MONGODB**
- **FIREBASE-ADMIN**
- **CORS**
- **DOTENV**

## How to Run

1. Clone the repository
2. Install dependencies with `npm install`
3. Set up environment variables (`.env` file):
   - `MONGODB_URI` for MongoDB connection
   - `FIREBASE_SERVICE_KEY` for Firebase Admin SDK
4. Start the server with `node server.js`
5. Server runs on `http://localhost:3000`
